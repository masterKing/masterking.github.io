解决git clone速度慢的问题

Created2018-10-26|Updated2023-10-19
|Word Count:216|Reading Time:1mins|Post Views:|Comments:

之前一直以为是公司对网络的封锁,导致git clone速度慢,后来网上查找这个问题的时候得知不能怪公司,是因为咱们在这个局域网内…
扯多了,回到正题,解决问题,由于我的是Mac电脑,我说的只针对Mac电脑

1. 进入终端命令行模式,输入

sudo vim /etc/hosts
